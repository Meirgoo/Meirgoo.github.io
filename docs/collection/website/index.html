website collect#

一些个人收藏的有意思/有用的站点收藏，或许很多站点都是可以直接在搜索引擎找到或者人尽皆知的，但这不重要，因为这里也作为我自己的站点收藏夹使用，说不准就能在这里找到你需要但是暂时不知道的呢。

电子书格式wiki#

在做电子书阅读相关的处理的时候就碰上一个棘手的问题，许多格式的定义并不全，就收藏了一个格式相关的东西，全英。

https://wiki.mobileread.com/wiki/Main_Page/zh

在线接短信#

前两年注册ChatGPT时候用的，后来注册Claude也能用，有些手机号已经关联过多了，价格很便宜，收一次验证码大约是0.2刀，不到两块，国内可以直接访问，没有正式的海外身份临时用一下很不错。

https://sms-activate.org/cn 

旧版Windows运行库#

对于某些使用 D3DX9、D3DX10、D3DX11、XAudio 2.7、XInput 1.3、XACT 和/或 Managed DirectX 1.1 的游戏，Microsoft DirectX® End-User Runtime 会安装许多来自旧版 DirectX SDK 的运行时库。

如果遇到“找不到d3d9x_xx.dll”等等类似问题可以试试这个。

安装流程最后有捆绑软件，微软官方还干这种事，这点差评。

https://www.microsoft.com/zh-cn/download/details.aspx?id=35

Google Font#

谷歌的字体图标等设计资源的公开网站，可以白嫖很多内容，我自己用了一些资源，根据协议是可以免费商用的，但不保证都符合这一协议，另外这个网站似乎不能在国内网络环境直接访问。

https://fonts.google.com/

搜索引擎收录#

谷歌搜索引擎站长控制台，在网站收录这方面，应该说谷歌和必应比其他好很多，一方面这两个搜索引擎流量够大，其次在提交难度和收录速度上远远低于国内其他网站，很多时候国内去处理一些个人的非常小的问题的时候就会面临这种情况：

要不要实名？要不要备案？要不要企业身份？

不做这些那就走不通，做这些吧，个人的没什么流量的小站点又完全不值，个人观点是，不想费大劲忙活做一个谷歌和必应就足够了。

https://search.google.com/search-console/

https://www.bing.com/webmasters/
