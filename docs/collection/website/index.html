      website collect #

      一些有意思/有用的站点收藏。

      资源站 #

      数学天地 #

      由Eric Weisstein创建的线上数学百科全书。

      中文站：https://mathworld.net.cn/

      英文站：https://mathworld.wolfram.com/

      音乐谱系图 #

      可视化的现代音乐历史和继承关系站点（全英）。

      https://musicmap.info/

      编程语言速查 #

      面向开发人员的大多数编程语言相关知识速查，不过普遍篇幅不长，并不怎么适合深入一门语言。

      源项目：https://github.com/Fechin/reference

      源站点：https://cheatsheets.zip/

      一个中文翻译站点（类似的有多个已部署项目）：https://quickref.cn/

      工具站 #

      哔哩哔哩视频下载 #

      新站好像支持抖音了，具体这个站点发展方向再看吧，我个人觉得这个站点前景不是特别明朗。

      https://xbeibeix.com/api/bilibili/

      在线接短信 #

      前两年注册ChatGPT时候用的，注册海外服务基本没什么问题，价格也很便宜，国内可以直接访问，体验下来是个还不错的网站。

      https://sms-activate.org/cn 

      游戏 #

      失落软件 #

      收集1965-2013失落游戏媒体的网站。

      https://www.myabandonware.com/

      nikoli 日本实体谜题提供商 （英日） #

      一家来自日本的“实体”谜题游戏提供商，书啊之类的，不得不说这家公司网站呈现很符合我对日本公司的刻板印象，页面底端怎么还2001-2021呢，实体游戏大概是没必要去玩了。

      https://www.nikoli.co.jp/en/

      PennyDell 美国实体谜题提供商 （英） #

      类似于上一条，不过是美国加拿大地区的，只能说海外实体游戏还是和大多数国人无缘了。

      https://www.pennydellpuzzles.com/

      Vinckensteiner 奥地利在线谜题 （英） #

      在线谜题，基本都是数独或者数独变种。

      https://raetsel.vinckensteiner.com/en/

      BrainBashers 英国在线谜题 （英） #

      在线谜题，种类多一些，但这个网站直观体验不太好，交互上一些东西可能不是那么直观。

      https://www.brainbashers.com/about.asp?25

      janko.at 德国(?)在线谜题 （德） #

      在线谜题，种类挺多，比较尴尬的是网站是德语，相对于英语可能会的少的多得多，并且好像部分谜题没有在线交互的方法，需要自己打印之类的，有在线交互的体验也比较一般。

      https://www.janko.at/Raetsel/index.htm

      康思 国际在线谜题 #

      在线谜题，有网站和APP，最大的优点是支持中文，谜题种类多，不过GUI感觉一般，页面元素可以看到都是图片堆起来的，并且好像不登陆只能玩每周谜题，登陆了有什么不一样不清楚。

      https://www.conceptispuzzles.com/zh/index.aspx

      推箱子online 在线推箱子专场 （英） #

      应该是一家日本公司，基本都是推箱游戏，在线解谜嵌套层数略多。

      https://www.sokobanonline.com/

      increpare games （英） #

      和前面的网站相比，这个网站离纯数学谜题（数独推箱之类）有点远了，而是更贴近现代电子游戏的在线形式，这个网站的作者或许名声不是特别大？但如果说最知名的作品史蒂芬的香肠卷，那喜欢解谜游戏的玩家应该多少都听说过。

      https://www.increpare.com/
